# Icons

Place your app icons in this directory. Required files:
- `32x32.png` - 32x32 pixel icon
- `128x128.png` - 128x128 pixel icon  
- `128x128@2x.png` - 256x256 pixel icon (for retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can generate these icons using tools like:
- [Icon Generator](https://www.iconfinder.com/icon-generator)
- [Favicon Generator](https://realfavicongenerator.net/)
- Or use an image editing tool to create them from a source image

For now, you can use placeholder icons or skip icon generation during development.
